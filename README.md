Implements HMD aim and movement for Re4 based on Praydog's reframework VR mod. Supports playing with a gamepad.

Install the latest version of Praydog's Reframework VR mod. Place re4_vr_hmd_aim_movement.lua under reframework\autorun under your RE4 game dir. Ensure reframework's re4_vr_crosshair.lua exists for this to work.
